# ☕ Interrogation (Interrogatório)

## 📖 Sobre
Esta aplicação fará perguntas ao usuário, e de acordo com o número de respostas positivas, o programa dirá se a pessoa é inocente, suspeita(o), cúmplice ou assassina(o). Foi feita utilizando apenas a linguagem `Java` e a IDS `VSCODE`.

<br>

## 📚 Instrução de Instalação

### 📋 Pré-requisitos
- [JDK 21.0.2](https://www.oracle.com/java/technologies/javase/jdk21-archive-downloads.html).

<br>

## 👩‍💻 Tecnologias Utilizadas

- Java - Versão: 21.0.2.
- Para a criação do aplicativo: [Visual Studio Code](https://code.visualstudio.com/).
